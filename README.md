# Vibe

AI가 만든 코드를 ‘검증·통제·개선’하는 백엔드 시스템

## 1. 프로젝트 개요

AI(예: OpenAI)가 생성한 코드를 그대로 사용하는 대신,

**백엔드 서버가 중간에서 코드를 검증·통제·개선 제안**하는 시스템을 구현한다.

본 프로젝트는 **바이브 코딩 시대의 AI 활용 방식에 대한 문제의식**을 바탕으로,

AI를 **신뢰 대상이 아닌 관리 대상**으로 다룬다.

## 2. 시스템 흐름도
```
[사용자]
   ↓ (AI가 생성한 코드 붙여넣기)
[백엔드 서버]
   ↓
1. 코드 정적 검증 (규칙 기반)
   ↓
2. OpenAI에 코드 분석 요청
   ↓
3. AI 응답 구조화 (JSON)
   ↓
4. 위험 요소 / 개선 포인트 판별
   ↓
[결과를 사용자에게 반환]
```
